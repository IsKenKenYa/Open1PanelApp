# 仪表盘模块索引

## 模块定位

仪表盘模块是Open1PanelApp的P0核心模块，作为用户登录后的首页，提供系统运行状态概览、资源使用情况、快捷操作入口等核心功能，是用户了解系统状态的主要入口。

## 子模块结构

| 子模块 | 端点数 | API客户端 | 说明 |
|--------|--------|-----------|------|
| **系统概览** | 4 | dashboard_v2.dart | 系统基本信息展示 |
| **资源监控** | 4 | dashboard_v2.dart | CPU、内存、磁盘、网络监控 |
| **快捷操作** | 2 | dashboard_v2.dart | 常用功能快捷入口 |
| **通知中心** | 2 | dashboard_v2.dart | 系统通知和告警展示 |

## 后续规划方向

### 短期目标
- 完善系统概览卡片
- 实现实时资源监控图表
- 添加快捷操作入口

### 中期目标
- 自定义仪表盘布局
- 添加更多监控指标
- 实现告警通知展示

### 长期目标
- 支持多服务器仪表盘
- 实现数据趋势分析
- 提供智能运维建议

## 与其他模块的关系

- **监控管理**: 共享监控数据
- **主机管理**: 获取主机信息
- **应用管理**: 应用状态展示
- **容器管理**: 容器状态展示
- **网站管理**: 网站状态展示
